# Expressive-Voice-Interface-for-AAC
This project enhances AAC communication by adding emotional expressivity and backchanneling to a TTS system. Users can generate speech with tones like neutral or happy and use quick responses via a Gradio UI. It improves conversational flow, reduces response time, and supports more natural interaction for people with speech impairments.


# 🎤 Chatterbox Turbo AAC

An expressive Augmentative and Alternative Communication (AAC) voice interface powered by Text-to-Speech (TTS), designed to improve natural interaction for users with speech impairments.

## Overview

This project explores how **emotional expressivity** and **backchanneling** can enhance communication in AAC systems.

It integrates the **Chatterbox-Turbo TTS model** with emotion control using reference audio, allowing users to generate speech with expressive tones such as:

- Neutral  
- Happy  
- Sleepy  

Additionally, the system provides **quick-access backchanneling phrases** (e.g., *“yeah”*, *“uh-huh”*) to support smoother conversational flow.

##Key Features

-  Emotion-controlled speech synthesis using reference audio  
-  Quick backchannel responses for real-time interaction  
-  Reduced cognitive load during conversations  
-  Improved conversational rhythm and engagement  
-  Lightweight UI built with Gradio
